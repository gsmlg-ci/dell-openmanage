# dell-openmanage

[![Build](https://github.com/gsmlg-ci/dell-openmanage/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/dell-openmanage/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/dell-openmanage`
- `ghcr.io/gsmlg-dev/dell-openmanage`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/dell-openmanage:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/dell-openmanage:latest
```

## Build

```bash
docker build -t gsmlg/dell-openmanage .
```
